# Test Plan – Mail Client (SPA)

## Testowany projekt
https://github.com/awaw92/mail

## Cel testów
Sprawdzenie poprawności działania aplikacji pocztowej SPA, w szczególności:
- wysyłania i odbierania wiadomości
- działania skrzynek (Inbox, Sent, Archive)
- dynamicznego ładowania danych przez API
- oznaczania wiadomości jako przeczytane

## Zakres testów
- logowanie i dostęp do skrzynek
- wysyłanie wiadomości
- przeglądanie wiadomości
- archiwizacja i odarchiwizowanie
- odpowiedzi na wiadomości

## Typy testów
- testy funkcjonalne
- testy API (REST)
- testy UI (SPA)
- testy integracyjne

## Środowisko
- przeglądarka Chrome
- desktop
